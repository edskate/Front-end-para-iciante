Está curioso com a sigla HTML?
Ela significa Hypertext Markup Languagem de marcação para hiper texto).
Ou, em uma tradução bem aberta, uma linguagem que possui tags para marcar documentos do tipo hipertexto.
Documento hipertexto é um que pode ter links para outros.
O HTML sozinho nada mais é que uma forma de marcação (através das tags) .
