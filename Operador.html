Operador Igual.

O operador igual (=) não funciona exatamente como na matemática.

Por exemplo, 2012 = ano não funciona, não é o mesmo que ano = 2012.

Dizemos que o operador = atribui o valor 2012 à variável, que fica a esquerda do =.

Repare que o uso das aspas define o que será impresso.

Se não há aspas dentro dos parenteses do document.write(...), o JavaScript buscará o valor daquela variável.

Caso contrário, utilizará o que está dentro das aspas apenas como uma sequência de caracteres (string), como vimos anteriormente.

É muito importante você mesmo testar e ver essa diferença
